# Echo Systems

A modern React-based web application built with Vite, featuring interactive components and smooth animations.

## Features

- User Authentication
- Dashboard Interface
- Voice Capture Functionality
- Neural Network Visualizations
- Responsive Design with Bento Grid Layout

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/subinedward/echo-twin.git
   cd echo-systems
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run the development server:
   ```
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Build

To build the project for production:

```
npm run build
```

## Preview

To preview the production build:

```
npm run preview
```
